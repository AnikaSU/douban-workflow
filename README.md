# douban-workflow

因为网上找到的大多数豆瓣的 workflow 都因为豆瓣 api 的关闭而失效，所以，自己动手，丰衣足食。

支持搜索书籍以及电影。

分别使用 `dbb` (douban book 简称)与 `dbm`(douban movie 简称)
